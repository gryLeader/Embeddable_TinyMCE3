Example of new plugin.

Instructions
------------
Copy all files to main folder. 
Copy the contents of the file en.js (it contains translatios for the plugin) into the file en.js of the main folder (append it to the end not overwriting anything).
Add the plugin in the TinyMCE init, in the plugins list and in the toolbar buttons (i.g.: cut, copy, paste, example, ..)
Run TinyMCE and you should see this plugin's button in the toolbar.
